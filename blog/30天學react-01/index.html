By Luna 01 January 0001

為什麼要寫這個主題？

我是剛接觸前端程式沒多久的菜鳥，在學完基本的 HTML、CSS、JavaScript 後，因為現在業界大部分都有在用前端框架，所以就想了解更多，就選擇 React 作為我第一個學習的前端框架，我目前的學習是以這本 React 「學習手冊-第二版-建構現代的 React 應用」為大綱，協助我可以更好的學習 React，以及該如何在網路搜尋與篩選資料，藉由寫文章的方式，希望可以內化為自己的東西，也因為我剛接觸程式沒多久，有些 JavaScript 的語法及方法運用的不是很好，這本書的一些內容對我這個 React 全然的新手來說，可以讓我更了解 React。

React 與 JavaScript 的關係？

在一開始剛接觸程式時，我一直不是很了解什麼是前端框架，他跟 JavaScript 差在哪？ 直到學到 JavaScript 的事件，寫了一個小小的購物車，光找節點就很麻煩還要生成元素、監聽事件，寫完程式碼真的是好長好醜，如果有要改的地方光是找就要花很久，不能想像要是東西很多那程式碼該有多可怕，也因此當我看到 React 不用找節點，只專注狀態和將狀態渲染到 HTML 上，而且我一次只處理部分組件，不用整個頁面一起處理，覺得真的是太棒了！也終於了解為何會有前端框架的出現。
